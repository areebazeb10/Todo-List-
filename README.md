# Advanced To-Do List 📝

A feature-rich To-Do List web application built using **HTML, CSS, and Vanilla JavaScript**. This project demonstrates DOM manipulation, LocalStorage usage, and dynamic UI rendering.

## 🚀 Features

* Add new tasks
* Set due date for each task
* Edit existing tasks
* Delete tasks
* Mark tasks as completed
* Dark mode toggle 🌙
* Persistent storage using LocalStorage 💾
* Clean and responsive UI

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)
* LocalStorage API

## 📸 Preview

Simple and clean task manager with dark mode, due dates, and task controls.

## 📂 Project Structure

```
index.html
```

## ⚙️ How It Works

* Tasks are stored in **LocalStorage**
* UI updates dynamically using JavaScript
* Each task includes:

  * Task text
  * Due date
  * Completion status
